予告編

作品情報

-イントロダクション-

同名絵本を原作として製作。ジャングルをテーマにしたボードゲームの物語。プレイヤーのもとに本物の動物たちが魔法のように現れる！？「ジュラシック・パークIII」などで知られるジョー・ジョンストン監督をはじめ、主演にはロビン・ウィリアムズ。 豪華キャスト陣が送るアドベンチャー・ファンタジー映画！

-ストーリー-

人気のない深夜の森で、急き込んだ様子の２人の少年が大きな木箱を地中深くに埋めていた。それから１００年後の１９６９年。気の弱い少年アランが、工事現場の土中から大きな木箱を発見する。その「ジュマンジ」と書かれた木箱は、何と大昔のゲーム盤だった。ガールフレンドのサラと共にゲームを開始したアランだったが、このゲームはダイスの目で進んだ箇所に書かれている事件が、実際に起きてしまうと云うとんでもない代物だった。それから更に２６年後、かつてアランが住んでいた屋敷にやって来た幼い姉妹のジュディとピーターは、屋根裏部屋で「ジュマンジ」を発見、ゲームを開始すると、吸血蚊や猿、ライオンなどと共に一人の汚い風貌の中年男性が出現、その人物とは、２６年前のゲームの最中に行方不明になったアランの成長した姿だったーー

レビュー

ボードゲームで遊んでいるとサイコロの出た目に止まったメッセージ通りの不思議なことが巻き起こる。 子供にウケが良さそうで大人も楽しめる内容のため、家族連れで映画館で観るのに最適。ビックリ箱のような映画。

夢中になって何も考えずに見れる映画です。 エンターテイメントとして、映画として、本当に優れたものだと思います。

作品のアイデアとしてはすごい良かったと思う。わかりやすくてシンプルで、ちゃんと落として欲しいところに落ちて、誰とでも安心して楽しめる映画。

キャスト&スタッフ

-キャスト-

アラン・パリッシュ / ロビン・ウィリアムズ

アメリカ合衆国の俳優・コメディアン。88年「ポパイ」で本格的映画デビュー。87年には、「グッドモーニング・ベトナム」ＤＪ役でアカデミー主演男優賞にもノミネートされ、映画スターとしての地位を確立。コメディを得意とする一方、「いまを生きる」「フィッシャー・キング」などシリアスな作品でも評価が高く、97年「グッド・ウィル・ハンティング」でオスカーを獲得する。 代表作は「レナードの朝」「パッチ・アダムス トゥルー・ストーリー」など。

サラ・ウィットル / ボニー・ハント

アメリカ合衆国の女優・声優。イリノ イ州シカゴ出身。シカゴの有名な劇団セカンド・シティで舞台女優として活躍。88年「レインマン」で映画デビュー。多くのハリウッド作品に出演。ＴＶ「Ｔｈｅ　Ｂｏｎｎｉｅ　Ｈｕｎｔ　Ｓｈｏｗ」では企画・脚本もこなし、2000年、デイビッド・ドゥカブニーとミニー・ドライバー主演のロマンチックコメディ「この胸のときめき」で監督デビューも果たした。 代表作は「夏休みのレモネード」「Dearフレンズ」など。

ジュディ・シェパード / キルスティン・キャロライン・ダンスト

アメリカ合衆国の女優。幼い頃からコマーシャルに出演し、89年「ニューヨーク・ストーリー」のウディ・アレンのエピソードで映画デビュー。「インタビュー・ウィズ・ヴァンパイア」での吸血鬼少女役で強烈な印象を残す。「ヴァージン・スーサイズ」のラックス役で女の子から絶大な人気を得る。 代表作は「インタヴュー・ウィズ・ヴァンパイア」「若草物語」など。

ピーター・シェパード / ブラッドリー・ピアース

アメリカ合衆国の俳優・声優。1982年10月23日生まれ。6歳のとき演技を始めるために南カリフォルニアに移住し、商業やテレビ、映画まで様々なプロジェクトに登場している。 代表作は「チャーリー」「クライズ・フロム・ザ・ハート」など。

-スタッフ-

監督 / ジョー・ジョンストン

アメリカのVFXアーティスト、映画監督。ILMの創設時からのメンバーで、『スター・ウォーズ エピソード4/新たなる希望』（1977年）で特殊効果に参加し、映画界入りした。『新たなる希望』ではカメオ出演もしており、デス・スターのドッキング・ベイ付近で活動するスペース・トルーパーと、スーパーレーザーのシャフトで管制を行う砲手の二役を演じている。主な監督作品は「ジュラシック・パークIII」「キャプテン・アメリカ/ザ・ファースト・アベンジャー」など。

脚本家 / ジョナサン・ブレア・ヘンズリー

アメリカ合衆国・マサチューセッツ州ミドルセックス郡出身の脚本家、映画監督、映画プロデューサー。テレビドラマ『インディ・ジョーンズ/若き日の大冒険』からキャリアをスタートさせ、1995年、『サイモン曰く』（Simon says）というオリジナルのシナリオが『ダイ・ハード3』の脚本に採用され、注目を集めた。以降、話題作の脚本のみならず、製作総指揮、監督としても活躍している。 代表作は「パニッシャー」「NEXT -ネクスト-」など。
